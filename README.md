# CPE_112B-Inventory-system
CPE_112B final project, A working inventory system that can create, read, update, and delete the data and information inside the system. This is a group project by Rosal, Leyson, Racal, Purisima, And Alicaba. 

Finishing up basic functionality for editing content of an inventory cell. Will be releasing a demo soon, in which the edit inventory function will be able to do the following:

- Edit a key.
- Edit a value of a key.
